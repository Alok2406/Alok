Two Methods can be used 1.Change Password 2.Create New SuperUser Method_1.Change Password of SuperUser Run below Step_1:-python.exe .\manage.py changepassword (UserName, for Ex. alok) Step_2:-Enter and confirm new Password Step_3:-done Password Change successfully. Method_2.Create New SuperUser Step_1:- python.exe .\manage.py createsuperuser Step_2:- Give Username:(for ex. alok) Step_3:- Give Email address:(for ex. alok@gmail.com) Step_4:- Give Password: and confirm it Step_5:- if Password enter is too small u get security notification Bypass by pressing (y) Superuser created successfully.
